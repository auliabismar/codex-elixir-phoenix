# Phoenix Plan

This skill handles the creation of comprehensive implementation plans for codebase modifications.

## Instructions

1. Clarify the requested outcome, constraints, and affected scope before planning.
2. Produce an ordered implementation plan that includes execution steps, risks, and validation targets.

## Usage Examples

- `$phx-plan "Add user authentication to the admin area"`
