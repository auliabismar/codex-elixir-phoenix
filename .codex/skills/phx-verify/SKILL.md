# Phoenix Verify

This skill handles the automated verification of implemented changes using the host system's testing facilities.

## Instructions

1. Select the most relevant verification commands for the current change.
2. Run or describe the checks, then report pass/fail results with any remaining risks or follow-up actions.

## Usage Examples

- `$phx-verify "Run the relevant tests for the latest change"`
