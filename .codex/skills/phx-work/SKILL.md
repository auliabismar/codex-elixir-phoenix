# Phoenix Work

This skill handles the sequential execution of tasks defined in an implementation plan.

## Instructions

1. Read the current plan or request and identify the next actionable implementation step.
2. Execute work sequentially, report progress clearly, and stop when blocked by ambiguity or failed verification.

## Usage Examples

- `$phx-work "Implement the next step from the current plan"`
